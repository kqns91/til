# ブロックチェーン

## ブロック

- prev_hash: 前のブロックのハッシュ値
- nonce: ナンス
- timestamp: タイムスタンプ
- transactions: トランザクション

## トランザクション

- sender: 送信者
- recipient: 受信者
- value: 送信額

## チェーン

- transaction_pool: トランザクションのリスト
- chain: ブロックのリスト

## マイニング

- ブロックのハッシュ値が特定の条件を満たすまで、nonceを変えながら計算を繰り返す
- ブロックのハッシュ値が条件を満たすと、ブロックを追加する
- ブロックを追加すると、トランザクションプールを空にする
